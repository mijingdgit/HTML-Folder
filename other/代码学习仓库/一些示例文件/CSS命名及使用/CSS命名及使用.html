1.容器

语义 命名 组合举例

盒子 box news-box / side-box

样式表拆分

<link rel="stylesheet" href="reset.css">
<link rel="stylesheet" href="public.css">
<link rel="stylesheet" href="main.css">
...
作用 命名 使用场景
重置 reset.css 清除系统默认样式
公共模块 public.css / common.css / base.css 大部分页面共同使用的样式
主要的 main.css / master.css / style.css 小网站使用一个样式文件即可
主题 themes.css 换肤
布局，版面 layout.css
表单 forms.css 所有表单相关的样式
打印 print.css 有些页面需要打印，单独写打印页的样式
... ... ...

<style>

    /*  放在原来的 .news-attr-read{} 后面，根据就近原则，能轻易维护 */

    .news .news-attr-read {}
