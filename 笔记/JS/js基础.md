# JavaScript
**JS是面向过程的语言而不是面向对象的语言**

    面向过程的语言三大特征：顺序，选择和循环
    面向对象的语言三大特征：封装，继承和多态
    面向过程的语言没有面向对的语言三大特征（如果有就是模拟的），但是面向对象的语言有面向过程的三大特征
